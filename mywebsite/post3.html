Django Practice Project

This project was developed while studying Django through the course.

During the course, we've studied many branches of the Django framework, such as templates, database, admin panel, CRUD, user login, flash massages, restricted pages, user registration and many others.

All the HTML/CSS templates were provided by the course for better concentration on Django itself.

The application aims to create a supportive community of programmers where everyone can help each other to become the better version of themselves.
